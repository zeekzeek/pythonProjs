filenameCounterBatch program usage

1. Download python from https://www.python.org/downloads/
2. Open 'Terminal'
3. Type in 'python --version'
4. Python version should be showing up on Terminal. If not, Python library is not in machine.
5. Acquire filenameCounterBatch.py
6. Navigate to folder where the above file is stored. To learn how to navigate Terminal, do a read up on https://www.macworld.com/article/221277/command-line-navigating-files-folders-mac-terminal.html [just take note on 'cd', using the 'Tab' key to complete filenames and 'ls']
7. Once you are in the folder, type 'python filenameCounterBatch.py'
8. Program should run.
9. A finder window should open, navigate to folder and select folder to begin batch count.
